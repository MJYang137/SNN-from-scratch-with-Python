## Requirements
本書で使用したPythonとライブラリ（のバージョン）一覧です
- Python >= 3.5
- numpy == 1.16.4
- matplotlib == 3.1.0
- tqdm == 4.32.2
- scipy == 1.3.0
- chainer == 6.0.0rc1

## Usage
### SingleFileSimulations
1ファイル完結のシミュレーションのコードです

### TrainingSNN
`./TrainingSNN/Models/`下のファイル内に書かれたモデルを用いてネットワークの構築と学習をシミュレーションします。
